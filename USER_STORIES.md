# User Stories

## Admin User Stories

### User Management
1. **As an admin**, I want to create, edit, and delete users (students, tutors, admins) so that I can manage the platform's user base.
2. **As an admin**, I want to view all users with their roles and status so that I can monitor user activity.
3. **As an admin**, I want to activate/deactivate user accounts so that I can control access to the platform.
4. **As an admin**, I want to assign tutor roles to users so that they can create and manage courses.

### Category Management
5. **As an admin**, I want to create, edit, and delete categories so that courses can be organized.
6. **As an admin**, I want to set category images and descriptions so that categories are visually appealing.
7. **As an admin**, I want to reorder categories so that important categories appear first.

### Course Management
8. **As an admin**, I want to view all courses in the system so that I can monitor course offerings.
9. **As an admin**, I want to approve or reject course publications so that only quality courses are available.
10. **As an admin**, I want to feature courses so that important courses get more visibility.
11. **As an admin**, I want to view course statistics (enrollments, ratings, reviews) so that I can assess course performance.

### Enrollment Code Management
12. **As an admin**, I want to view all enrollment codes so that I can track code usage.
13. **As an admin**, I want to generate enrollment codes for tutors so that they can enroll students.

### Analytics & Reports
14. **As an admin**, I want to view platform statistics (total users, courses, enrollments) so that I can understand platform growth.
15. **As an admin**, I want to view tutor activity so that I can monitor tutor engagement.
16. **As an admin**, I want to view student progress across all courses so that I can identify struggling students.

## Tutor User Stories

### Course Creation & Management
17. **As a tutor**, I want to create courses with title, description, and learning outcomes so that I can share my knowledge.
18. **As a tutor**, I want to upload course images so that my courses are visually appealing.
19. **As a tutor**, I want to set course level (beginner, intermediate, advanced) so that students know the difficulty.
20. **As a tutor**, I want to add course modules with titles and descriptions so that I can organize course content.
21. **As a tutor**, I want to add topics to modules with videos, transcripts, and descriptions so that students can learn effectively.
22. **As a tutor**, I want to upload course resources (documents, PDFs) so that students have additional learning materials.
23. **As a tutor**, I want to set course pricing or make it free so that I can monetize my courses.
24. **As a tutor**, I want to publish/unpublish my courses so that I can control course availability.

### Module & Topic Management
25. **As a tutor**, I want to reorder modules and topics so that content flows logically.
26. **As a tutor**, I want to add video URLs to topics so that students can watch video lessons.
27. **As a tutor**, I want to add transcripts to video topics so that students can read along.
28. **As a tutor**, I want to mark topics as free or paid so that I can control access.

### Enrollment Code Management
29. **As a tutor**, I want to generate enrollment codes for my courses so that I can enroll specific students.
30. **As a tutor**, I want to view all enrollment codes I've generated so that I can track code usage.
31. **As a tutor**, I want to set expiration dates for enrollment codes so that codes don't remain valid indefinitely.

### Assignment Management
32. **As a tutor**, I want to create assignments for my courses so that students can practice what they've learned.
33. **As a tutor**, I want to set assignment due dates and maximum scores so that students know expectations.
34. **As a tutor**, I want to view student assignment submissions so that I can grade them.
35. **As a tutor**, I want to grade assignments and provide feedback so that students can improve.
36. **As a tutor**, I want to see which students have submitted assignments so that I can track completion.

### Test Management
37. **As a tutor**, I want to create CBT (Computer-Based Tests) for modules so that students can test their knowledge.
38. **As a tutor**, I want to add multiple-choice, true/false, and short answer questions to tests so that I can assess different skills.
39. **As a tutor**, I want to set passing scores for tests so that students must achieve a minimum score to proceed.
40. **As a tutor**, I want to set time limits for tests so that students complete tests within a timeframe.
41. **As a tutor**, I want to view test attempts and scores so that I can assess student performance.

### Student Interaction
42. **As a tutor**, I want to view all students enrolled in my courses so that I can track enrollment.
43. **As a tutor**, I want to view student progress in my courses so that I can identify students who need help.
44. **As a tutor**, I want to receive messages from students so that I can answer their questions.
45. **As a tutor**, I want to send messages to students so that I can provide guidance.
46. **As a tutor**, I want to view student notes for my courses so that I can understand what students find important.

### VR & DIY Content
47. **As a tutor**, I want to add VR learning content to my courses so that students can have immersive learning experiences.
48. **As a tutor**, I want to add DIY (Do-It-Yourself) projects to my courses so that students can practice hands-on.
49. **As a tutor**, I want to set difficulty levels for DIY projects so that students know what to expect.

### Course Recommendations
50. **As a tutor**, I want to recommend related courses so that students can continue learning.

## Student User Stories

### Registration & Authentication
51. **As a student**, I want to register an account so that I can access courses.
52. **As a student**, I want to login to my account so that I can continue learning.
53. **As a student**, I want to view my profile so that I can see my information.

### Course Discovery
54. **As a student**, I want to browse all available courses so that I can find courses of interest.
55. **As a student**, I want to filter courses by category so that I can find courses in specific areas.
56. **As a student**, I want to filter courses by level so that I can find courses appropriate for my skill level.
57. **As a student**, I want to search for courses by title so that I can quickly find specific courses.
58. **As a student**, I want to view course details (description, tutor, reviews, modules) so that I can decide if I want to enroll.
59. **As a student**, I want to see recommended courses so that I can discover related content.

### Enrollment
60. **As a student**, I want to enroll in a course using an enrollment code so that I can access course content.
61. **As a student**, I want to enroll directly in free courses so that I can start learning immediately.
62. **As a student**, I want to view my enrolled courses so that I can access them easily.
63. **As a student**, I want to see my course progress so that I know how much I've completed.

### Learning Experience
64. **As a student**, I want to view course modules and topics so that I can navigate course content.
65. **As a student**, I want to watch video lessons so that I can learn from video content.
66. **As a student**, I want to read transcripts while watching videos so that I can follow along.
67. **As a student**, I want to mark topics as complete so that I can track my progress.
68. **As a student**, I want to see my progress percentage for each course so that I know how much I've completed.
69. **As a student**, I want to take notes while learning so that I can remember important points.
70. **As a student**, I want to view all my notes for a course so that I can review them later.
71. **As a student**, I want to edit and delete my notes so that I can keep them organized.
72. **As a student**, I want to download course materials so that I can access them offline.

### Assignments
73. **As a student**, I want to view assignments for my enrolled courses so that I can complete them.
74. **As a student**, I want to submit assignments with text and file uploads so that I can complete coursework.
75. **As a student**, I want to view my assignment submissions so that I can see what I've submitted.
76. **As a student**, I want to see assignment grades and feedback so that I can improve.

### Tests
77. **As a student**, I want to take module tests so that I can assess my knowledge.
78. **As a student**, I want to see my test scores so that I know how well I performed.
79. **As a student**, I want to see if I passed a test so that I know if I can proceed to the next module.
80. **As a student**, I want to see test explanations for correct answers so that I can learn from mistakes.

### VR & DIY
81. **As a student**, I want to access VR learning content so that I can have immersive learning experiences.
82. **As a student**, I want to view DIY projects with instructions so that I can practice hands-on skills.
83. **As a student**, I want to see materials needed for DIY projects so that I can prepare.

### Communication
84. **As a student**, I want to send messages to my course tutor so that I can ask questions.
85. **As a student**, I want to receive messages from my tutor so that I can get guidance.
86. **As a student**, I want to see unread message notifications so that I don't miss important messages.

### Certificates
87. **As a student**, I want to receive a certificate when I complete a course so that I can prove my achievement.
88. **As a student**, I want to view all my certificates so that I can showcase my accomplishments.
89. **As a student**, I want to download my certificates so that I can share them.

### Reviews & Ratings
90. **As a student**, I want to rate and review courses I've completed so that I can share my experience.
91. **As a student**, I want to read course reviews so that I can make informed enrollment decisions.

### Progress Tracking
92. **As a student**, I want to see a list of my ongoing courses so that I can continue learning.
93. **As a student**, I want to see a list of my completed courses so that I can review what I've learned.
94. **As a student**, I want to see my overall progress across all courses so that I can track my learning journey.

## System Requirements

### Performance
- System should handle 100,000+ concurrent users
- API response time should be under 200ms for most endpoints
- Database queries should be optimized with proper indexing
- File uploads should be handled efficiently

### Security
- All user passwords should be hashed
- API endpoints should be protected with authentication
- Enrollment codes should be unique and secure
- File uploads should be validated and sanitized

### Scalability
- System should support horizontal scaling
- Database should be optimized for large datasets
- Caching should be implemented for frequently accessed data
- CDN should be used for static assets

### User Experience
- API responses should be consistent and well-structured
- Error messages should be clear and helpful
- Loading times should be minimized
- Mobile-friendly API responses

## Technical Implementation Notes

### Database Optimization
- Indexes on frequently queried fields (user_id, course_id, status)
- Composite indexes for common query patterns
- Proper foreign key constraints
- Database query optimization

### API Design
- RESTful API design
- Consistent response formats
- Proper HTTP status codes
- Pagination for list endpoints
- Rate limiting to prevent abuse

### File Storage
- Secure file storage for course materials
- Image optimization for course images
- File type validation
- Virus scanning for uploaded files

### Notifications
- Email notifications for important events
- In-app notifications for messages
- Progress reminders
- Assignment deadline reminders
